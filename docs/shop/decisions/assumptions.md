# Working Assumptions (Shop Phase)

These assumptions guide implementation until explicitly revised.

## Narrative + Unlock

- Space Man encounter is a one-time unlock sequence.
- Shop becomes visible only after encounter completion.

## Economy

- Oxygen tanks are the primary shop currency.
- Currency is persisted locally first (offline-first).

## Upgrade Set (v1)

- Hazard pressure reduction
- Oxygen spawn improvement
- Tank value increase

## Delivery Constraints

- Keep locked epic order unless critical blockers force a change.
- Favor deterministic logic and maintainability over speed hacks.
